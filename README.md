# Tudy Company

`D:\workspace\tudy_company`는 TUDY 소개/회사 홈페이지를 위한 독립 프로젝트입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 구성

- `index.html`: 랜딩 페이지 마크업
- `styles.css`: 반응형 스타일
- `script.js`: 간단한 문의 폼 안내 동작

## 메모

- 현재는 정적 프로젝트로 구성했습니다.
- 기존 `D:\workspace\tudy` 앱과 분리되어 있어서 운영 기능이나 인증 구조에 영향을 주지 않습니다.
- 실제 문의 저장/메일 발송이 필요하면 여기 프로젝트에 API 연동을 추가하면 됩니다.
